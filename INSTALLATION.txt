Chrome:
1. Download source from https://github.com/Prozilla/Wooclap-extension/archive/refs/heads/main.zip
2. Unzip downloaded file
3. Open "chrome://extensions" in Chrome
4. Enable "Developer mode" in the top right corner
5. Click "Load unpacked" in the top left corner
6. Select the "src" folder

Firefox:
1. Download source from https://github.com/Prozilla/Wooclap-extension/archive/refs/heads/main.zip
2. Unzip downloaded file
3. Open "about:debugging" in Firefox
4. Click on "This Firefox" > "Load Temporary Add-on..."
5. Select "manifest.json" in the "src" folder
